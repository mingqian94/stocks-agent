# AI 量化炒股助手

## 项目概述

- **目标市场**: A股 + ETF（不含港股、美股）
- **核心策略**: 动量轮动（Momentum Rotation）
- **风险控制**: 最大回撤 < 30%
- **收益目标**: 年化 8%+
- **数据源**: 东方财富行情API、动量排名
- **系统架构**: 自动盯盘脚本 + 实时仪表盘 + 比赛雷达

## 二、核心策略：动量轮动

### 策略逻辑

1. **动量排序**: 监测22只ETF实时涨跌幅
2. **强者买入**: 买入涨幅排名前2且为正的ETF
3. **弱者减仓**: 持仓不在Top2且跌幅<-0.5%时减半仓
4. **止损保护**: 单日跌幅≥3%时减半仓

### ETF标的池（22只）

**宽基**（5只）：上证50/沪深300/中证500/中证1000/科创50
**科技**（3只）：半导体ETF/芯片ETF/科创芯片ETF
**金融**（2只）：证券ETF/银行ETF
**医药**（2只）：医药ETF/医疗ETF
**新能源**（3只）：新能源车ETF/光伏ETF/新能源ETF
**消费**（2只）：酒ETF/房地产ETF
**周期**（2只）：有色金属ETF/有色ETF
**军工/传媒**（3只）：军工ETF/传媒ETF/中概互联ETF
**其他**（2只）：其他ETF/

### 交易规则

| 规则 | 阈值 | 动作 |
|------|------|------|
| 动量Top2且为正 | 涨幅排名前2 | 平均分配可用资金买入 |
| 弱势减仓 | 不在Top2且跌幅<-0.5% | 减半仓 |
| 止损 | 单日跌幅≥3% | 减半仓 |

## 三、自动盯盘系统

### 脚本：auto_trade.py

**运行方式：后台持续运行

**检查频率**：每3分钟检查一次（交易时间）

**主要功能：
- 持仓查询（东方财富API）
- 行情获取（22只ETF实时涨跌幅）
- 动量排名计算
- 自动执行买卖决策
- 日志记录

### API配置

**东方财富模拟交易API**：
- API URL: `https://mkapi2.dfcfs.com/finskillshub/api/claw/mockTrading`
- API Key: 环境变量 `MX_APIKEY`

## 四、实时监控仪表盘

### 仪表盘：dashboard.py + templates/dashboard.html

**访问地址**：http://localhost:5000

**功能模块**：
1. **资产概览卡片**：总资产、累计收益、可用资金、持仓数量
2. **收益分析**：今日/本周/本月/累计 多维度收益分析
3. **持仓详情表格**：标的、成本价、现价、涨跌幅、盈亏、操作
4. **动量排名Top10**：实时显示22只ETF涨幅排名
5. **交易提醒区**：止损提醒、买入提醒
6. **操作日志**：自动记录所有交易和提醒
